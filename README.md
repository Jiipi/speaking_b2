# IELTS Speaking Guide - Band 5.0+

Trang tĩnh hướng dẫn luyện IELTS Speaking, viết bằng HTML/CSS/JS thuần (single-file).

## Xem trực tiếp

Mở [index.html](index.html) bằng trình duyệt, hoặc truy cập bản deploy trên Vercel.

## Deploy lên Vercel

1. Import repo này vào [vercel.com/new](https://vercel.com/new).
2. Framework Preset: **Other** (static site, không cần build).
3. Root Directory: `./`
4. Nhấn **Deploy**.

Vercel sẽ tự phục vụ `index.html` ở root.

## Cấu trúc

```
.
├── index.html      # Toàn bộ nội dung, style và script
├── vercel.json     # Cấu hình Vercel (tuỳ chọn)
├── .gitignore
└── README.md
```
